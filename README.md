# Anthaathi Agent

AI-powered coding assistant interface.

## Structure

```
├── src/          # React + TypeScript frontend
├── backend/      # Node.js API server
└── .env.example  # Environment template
```

## Quick Start

```bash
# Frontend
yarn install
yarn dev

# Backend
cd backend
yarn install
yarn dev
```

## Environment

Copy `.env.example` to `.env.local` and update values:

```bash
cp .env.example .env.local
```
